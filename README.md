# 📊 Sales Performance & Revenue Growth Dashboard

## 🚀 Project Overview
This project is an interactive Power BI dashboard designed to analyze **sales performance, revenue growth, and business KPIs**.  

It helps businesses monitor performance across regions, products, and sales channels, enabling **data-driven decision-making**.

---

## 🎯 Objective
To build a dynamic dashboard that provides insights into:
- Revenue and profit trends  
- Sales performance across regions and categories  
- Target vs Actual sales comparison  
- Business growth patterns  

---

## 🛠️ Tools & Technologies Used
- Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  
- Microsoft Excel  

---

## 📁 Dataset Description
The dataset contains 100+ records of sales transactions with the following fields:

- Date, Month, Quarter, Year  
- Region, City  
- Sales Executive  
- Product Category & Product Name  
- Units Sold, Selling Price  
- Revenue, Cost, Profit  
- Profit Margin %  
- Customer Type (New/Existing)  
- Sales Channel (Online/Offline)  
- Target
